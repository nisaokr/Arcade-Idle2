## ğŸ® Gameplay

Players engage in an interactive VR experience where they:

- **Navigate** the virtual environment using Meta Quest 2 XR controllers
- **Collect** apples by grabbing them from trees using intuitive grab mechanics
- **Aim and Throw** collected apples at designated targets (**TargetFront** and **TargetBack**)
- **Score Points** based on target difficulty and accuracy
- **Track Progress** via the real-time score display on the Canvas Score Board located next to the house

Each target has a different point value, encouraging players to develop precision and strategic throwing techniques.

## ğŸš€ Installation

### Clone the Repository

```bash
git clone https://github.com/nisaokr/Arcade-Idle2.git
cd Arcade-Idle2
```

### Open in Unity

1. Launch **Unity Hub**
2. Click **Add** and navigate to the cloned repository folder
3. Select the project to open it in Unity Editor
4. Wait for Unity to import all assets and dependencies

### Build and Deploy

1. Connect your Meta Quest 2 headset to your computer via USB
2. Enable **Developer Mode** on your Meta Quest 2 device
3. In Unity, go to **File > Build Settings**
4. Select **Android** as the platform
5. Configure build settings for Meta Quest 2 (VR settings)
6. Click **Build and Run** to deploy to your device

> **Note**: Ensure you have the Android SDK and Meta Quest development tools properly configured in Unity.

## ğŸ¯ How to Play

1. **Launch the Game** on your Meta Quest 2 headset
2. **Navigate** the virtual environment using your controllers
3. **Approach** the apple trees scattered throughout the scene
4. **Grab** apples by reaching out and using the grab controls on your controllers
5. **Aim** at the targets (TargetFront or TargetBack)
6. **Throw** the apples by releasing the grab control while making a throwing motion
7. **Monitor** your score on the Canvas Score Board
8. **Challenge** yourself to improve your accuracy and hit higher-value targets
